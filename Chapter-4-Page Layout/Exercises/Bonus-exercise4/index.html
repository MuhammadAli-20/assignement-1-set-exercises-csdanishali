About

This Study Planner is a professionally designed single-page web application created for students enrolled in the Web Development module of the Creative Computing course. The primary goal of this planner is to help students manage their study workload effectively by organizing tasks and providing access to useful academic information.

Module Overview

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web, such as headings, paragraphs, links, and images.

CSS (Cascading Style Sheets) is used to control the appearance of web pages by defining styles for elements such as layout, colors, fonts, and spacing.

JavaScript is a programming language that enables interactivity on web pages, allowing you to create dynamic content like form validation, animations, and real-time updates.
