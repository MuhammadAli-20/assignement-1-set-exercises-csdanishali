HTML and CSS quick-reference guide



HTML tags

HTML Headings <h1> to <h6>

Used to define headings. <h1> is the largest, <h6> the smallest.

<h1>Main Title</h1><h2>Subheading</h2>

HTML Paragraphs

The HTML <p> element defines a paragraph.

<p>This is a paragraph.</p>

HTML Images

The HTML <img> tag is used to embed an image in a web page.

<img src="car.jpg" alt="car image">

HTML Links

The HTML <a> tag defines a hyperlink.

<a href="https://www.youtube.com/">Visit Youtube</a>

HTML Bold Tag <b>

The HTML <b> makes the text bold.

<p>This is normal text - <b>and this is bold text</b>.</p>

HTML Italic Tag <i>

The HTML <i> makes the text italic.

<p><i>This is italic text.</i>is the most popular filler text in history.</p>

HTML <br> Tag

The HTML <br> tag inserts a single line break.

<p>br tag inserts a break line.<br></p>

HTML <div> Tag

The HTML <div> tag defines a division or a secction in an HTML document.

<div>
<p>This is some text in div element.</p>
</div>

HTML <ol> Tag

The HTML <ol> tag defines an ordered list.

<ol>
<li>Apple</li>
<li>Banana</li>
<li>Kiwi</li>
<li>Watermelon</li>
</ol>

HTML <ul> Tag

The HTML <ul> tag defines an unordered list.

<ul>
<li>Apple</li>
<li>Banana</li>
<li>Kiwi</li>
<li>Watermelon</li>
</ul>

CSS Properties

CSS Color

This CSS property changes the color of a text.

<h1
style="color: red;"
>Hello World!</h1>

CSS Background

This CSS property changes the background color of a text.

<h1
style = "background=color: blue;"
></h1>

CSS Margins

This CSS property is used to create space around elements, outside of any defined borders.

<p
style = "margin: 25px 50px 75px;"
></p>

CSS Padding

This CSS property is used to generate space around an element's content, inside of any defined borders.

<h1
style = "padding: 25px 50px;"
></h1>

CSS Height and Width

This CSS property is used to set the height and width of an element.

<div
style = "height: 30px; width: 30px;"
></div>

CSS Text Alignment

This CSS property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.

<h1
style = "text-align: center;"
></h1>
<h2
style = "text-align: left;"
></h2>
<h3
style = "text-align: right;"
></h3>

CSS Text Decoration

This CSS property is used to add a decoration line to text.

<p
style = "text-decoration: underline;"
></p>
<p
style = "text-decoration: overline;"
></p>
<p
style = "text-decoration: line-through;"
></p>

CSS Opacity

This CSS property is used to add a decoration line to text.

<p
style = "text-decoration: underline;"
></p>
<p
style = "text-decoration: overline;"
></p>
<p
style = "text-decoration: line-through;"
></p>

CSS Text Decoration

This CSS property specifies the opacity/transparency of an element.

<h3
style = "opacity: 1.0;"
></h3>
<h4
style = "opacity: 0.5;"
></h4>
<h5
style = "opacity: 0,7;"
></h5>
